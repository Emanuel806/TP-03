1. ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta ul (unordered list) sirve para crear listas no ordenadas, en la que la secuencia de los elementos no es importante.

Un ejemplo de utilización sería para elaborar una lista de deseos para el 2025, ya que cada uno surge de forma espontánea y no tienen un orden previsto para ser realizados.

2. ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

La etiqueta ol (ordered list) permite definir listas o viñetas ordenadas, ya sea con numeración o alfabéticamente. En este tipo de listas las secuencias de los elementos es importante, y estos tienen un criterio de ordenamiento.

Un ejemplo de utilización sería para elaborar una lista de escenas para una película o serie, en la que cada acción ocurre antes o después de otra y su orden es muy importante para la trama.

3. ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

La etiqueta dl (descriptive list) representa una lista descriptiva, que consiste en pares de elementos nombre-valor, incluidos, entre otros, términos y definiciones.

Un ejemplo de utilización sería para elaborar una lista de desarrollo de personajes para una película o serie, en la que cada personaje tiene un nombre y una descripción asignada.

4. ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

La etiqueta li (item list) declara cada uno de los elementos de una lista, ordenada o desordenada. Dentro de estas etiquetas se escribe el elemento de una lista, y a su vez, se escriben dentro de las etiquetas de la lista que conforman.

Un ejemplo de utilización sería para escribir nombres de actores o actrices famosos dentro de una lista ordenada según su popularidad.

9. Existe un atributo que se aplica a la etiqueta ol para ver la lista en orden descendente en vez de ascendente, ¿cuál es ese atributo? Utilizar la referencia de atributos de HTML.

El atributo "reversed", utilizado en listas ordenadas, indica si la lista debe ser mostrada en orden descendente en vez de ascendente.
